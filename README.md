# REGGAETRAVEL.COM
## Follow the rhythm. Honor the roots.

ReggaeTravel is a global cultural travel platform built on one core belief:

> **Reggae is a place, not a playlist.**

This is not “reggae vibes” tourism.  
This is not beach resorts playing Bob Marley at happy hour.

ReggaeTravel is a geography-first, history-first, community-first guide to the real places where reggae was born, where it traveled, and where it still lives.

You don’t consume reggae.  
You travel through it.

---

## What ReggaeTravel Is

ReggaeTravel is a **cultural cartography mega-site** that maps reggae as:

- history
- migration
- neighborhood culture
- sound system infrastructure
- studios and producers
- festivals and rituals
- diaspora networks
- liberation movements
- joy, grief, and spiritual survival

ReggaeTravel exists to help travelers experience reggae **with reverence, context, and respect**.

## The Roots → Routes Framework (Core Site Architecture)

This is the signature organizing principle.

### ROOTS (Origins)
Where reggae comes from:

- ska → rocksteady → reggae
- Rastafari and spiritual tradition
- political context and liberation history
- sound system culture as community infrastructure
- patois as living language
- studios, producers, and recording sites

### ROUTES (Where it Traveled)
Where reggae moved through migration and diaspora:

- UK (London, Birmingham, Bristol)
- Canada (Toronto)
- USA (New York, Miami)
- Africa (Lagos, Accra, Addis Ababa)
- Japan (Tokyo — a major global node)
- Europe (Berlin, Paris)
- South America (São Paulo)

This turns reggae into a map — not a genre.
